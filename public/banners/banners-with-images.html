How to Export These Banners:

  1. Open this file in browser - Double-click the file or open with browser
  2. Wait for images to load - The images are from Unsplash and will load automatically
  3. Take screenshot of each banner:
    • Windows: Use Snipping Tool (Win+Shift+S) and select each banner
    • Mac: Press Cmd+Shift+4 and drag to select
    • Browser: Press F12 → Elements tab → Right-click banner div → "Capture node screenshot"
  4. Save as PNG: Save each with names:
    • banner-1-first-order.png
    • banner-2-bridal.png
    • banner-3-express.png
  5. Upload to Admin Panel: Go to Admin → Settings → Banners and upload each image

Tip: The images used are high-quality photos from Unsplash showing real makeup artists and brides. You can replace them with your own images by editing the CSS background URLs.
