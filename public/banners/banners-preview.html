How to use these banners:

  1. Screenshot each banner - Right-click on each banner and select "Save image as" (or use screenshot tool)
  2. Go to Admin Panel - Navigate to Admin → Settings → Banners
  3. Upload Images - Click "Add Banner" and upload each image
  4. Configure Settings:
    • Banner 1 (Discount): Title="20% OFF", Subtitle="On your first booking", Button="Book Now", Link="/offers"
    • Banner 2 (Bridal): Title="Bridal Makeup", Subtitle="Look perfect on your special day", Button="Explore", Link="/bridal"
    • Banner 3 (Quick): Title="Quick Book", Subtitle="Same-day appointments available", Button="Book Now", Link="/quick-book"
  5. Set Active Dates - Choose when each banner should be displayed
  6. Save and Activate - Enable the banners
